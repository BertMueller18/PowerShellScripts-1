<h2>What is Autodiscovery?</h2>
<p>Exchange Autodiscover is a web service that helps Microsoft Exchange administrators configure user profile settings for clients running Outlook 2007 or later, and mobile phones running Windows Mobile 6.1 or later.</p>
<p>In order for autodiscovery to work properly, DNS must be configured in a right way:</p>
<ul>
<li>DNS must contain CNAME record autodiscover.yourdomain.com</li>
<li>Autodiscover record must point to A record of yourdomain.com</li>
</ul>
<p>Autodiscover can be tested via Microsoft Exchange Connectivity Analyzer on <a href="https://testconnectivity.microsoft.com/">this</a> link</p>
<p>If it is set up in a proper way Cloud Migration Platform will show all mailboxes on which user whose credentials were used during logon has full mailbox access or impersonation rights set up.</p>
<p>Learn more about <a href="http://devtech.kayako.com/Knowledgebase/Article/View/18/11/setting-up-full-mailbox-access">full mailbox access</a> or <a href="http://devtech.kayako.com/CloudMigrationPlatform/Knowledgebase/Article/View/17/11/impersonation-rights-setup">impersonation rights</a> setup.</p>
